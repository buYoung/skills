# Competitive Analysis

Framework for mapping competitive landscape and identifying market positioning opportunities.

## Competitor Categories

| Category | Definition | Analysis Depth |
|----------|------------|----------------|
| Direct | Same product, same customer segment | Full analysis |
| Indirect | Different product, same problem solved | Positioning comparison |
| Potential | Could enter market with resources | Threat assessment |

## Analysis Dimensions

### Market Position Matrix

| Dimension | Metrics |
|-----------|---------|
| Market Share | Revenue %, customer count, transaction volume |
| Growth Rate | YoY revenue growth, customer growth |
| Brand Strength | Recognition, NPS, social presence |

### Competitive Attributes

| Attribute | Data Points |
|-----------|-------------|
| Pricing | Price points, pricing model, discounting |
| Product | Features, quality, differentiation |
| Distribution | Channels, geographic reach, partnerships |
| Marketing | Positioning, messaging, ad spend |
| Technology | Platform, innovation, IP |

### Porter's Five Forces Summary

| Force | Assessment Criteria |
|-------|---------------------|
| Competitive Rivalry | Number of competitors, industry growth, differentiation |
| Supplier Power | Supplier concentration, switching costs |
| Buyer Power | Buyer concentration, price sensitivity, alternatives |
| Threat of Substitutes | Substitute availability, switching costs |
| Threat of New Entrants | Barriers to entry, capital requirements, regulations |

## Competitive Moat Assessment

| Moat Type | Indicators |
|-----------|------------|
| Network Effects | Value increases with users |
| Switching Costs | High cost/effort to change |
| Cost Advantage | Lower unit economics than competitors |
| Intangible Assets | Patents, brand, regulatory licenses |
| Efficient Scale | Natural monopoly dynamics |

## Output Format

```
Competitive Intensity: Low / Medium / High
Market Concentration: Fragmented / Moderate / Concentrated
Key Competitors: [List with market share]
Differentiation Opportunity: [Description]
Moat Potential: Weak / Moderate / Strong
Entry Barriers: Low / Medium / High
```

## Data Collection via Web Search

Use web search to gather competitive intelligence:

| Search Query Examples | Target Data |
|-----------------------|-------------|
| "[competitor name] funding series" | Competitor financial strength |
| "[competitor name] pricing plans" | Pricing comparison |
| "[industry] market share leader 2024" | Market position mapping |
| "[competitor name] reviews complaints" | Weakness identification |
| "alternatives to [competitor]" | Indirect competitor discovery |

| Category | Sources |
|----------|---------|
