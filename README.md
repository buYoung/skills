# Initialize Repo

**Initialize Repo** is a tool that automatically sets up the basic structure and configuration required when starting a new project. It specifically focuses on establishing an environment where developers and AI can communicate efficiently by generating metadata files such as `AGENTS.md` for collaboration with AI agents.

- Separation of Behavior and Capabilities:
    - AGENTS.md: Defines the AI agent's behavior, rules, and workflows.
    - SKILL.md: Defines tools (capabilities) and skills usable by the agent (Compatible with Claude Skills).
- Project Scaffolding: Generates optimized directory structures for different project types such as Web, CLI, and Library.
- Development Environment Automation: One-click setup for essential configurations like Git initialization (.gitignore) and Linter/Formatter (ESLint, Prettier).
